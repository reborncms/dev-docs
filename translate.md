#Translate Class for Reborn CMS.

##How to use Translate Class

###setPath()

Set the Language File Path.

Default path are [ CORE_MODULES, MODULES, THEMES, ADMIN_THEME ]

* *(array)* Path array for language file.

**Sample Code**

	\Translate::setPath($path);
	
	
###load()

waiting

* *()* resource

* *()* local

* *()* type

**Sample Code**

	\Translate::load();
	
	
###get()

waiting

* *()* Key

* *()* default

* *()* Local

* *()* Type

**Sample Code**

	\Transloate::get();
